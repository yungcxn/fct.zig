# fct.zig

My own functional programming lib in zig.

This is just for educational and experimental purposes, and it is much rather recommended to stick to zig's way of coding, rather than the way advertised through the code in this repo.

But if you still insist on functional programming, that's for you.

# Test

There is no need to build, but you could build and run all tests:

```bash
zig build test
```

# Installation

Copy the files in `fct/` in the folder you like and start importing.
